# Aditya Dosapati Portfolio

Live Website: https://aditya-dosapati.github.io/Personal-Portfolio-Website/

Neo-brutalist personal portfolio built with Next.js, React, TypeScript, and Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Main Files

- src/data/portfolio.ts (content)
- src/app/page.tsx (layout)
- src/app/globals.css (styling)

## Deploy

GitHub Pages deploy is automated via .github/workflows/deploy-pages.yml.
